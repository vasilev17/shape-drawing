# Shape Drawing

This is a WinForms desktop application that enables the user to draw, edit, move, calculate area, open and save shape drawings.

## Installation

Run the `"ShapeDrawing.sln"` file.

## Usage

### Main controls:

<pre>
- <b>Draw</b> a shape: Click <b>Middle Mouse Button</b> (scroll button)
- <b>Edit</b> a shape: Click <b>Right Mouse Button</b> (on a shape)
- <b>Move</b> a shape: <b>Left Mouse Button</b> (drag and drop)
</pre>
